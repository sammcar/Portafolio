# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, no harassment of any kind.

Instances of unacceptable behavior may be reported to the repository owner
via the contact information on the published site. All reports will be
reviewed and handled with discretion.
